# fld2csv.py

Parser for FLD files to CSV format.


## Usage

```fld2csv.py <input file> -o <outputFile> -t <type>```

Where -o is NOT specified, it will be renamed to fileName.csv .


Where -t is NOT specified, it will *always* default to fld.
